Q: What is an edge list?  
A: A collection of edges (connections) which represent a graph.  
e.g.
```javascript
// a third element can be inserted into each array to apply a weight graph e.g. [0, 2 ,88]
const graph = [
  [0, 2],
  [2, 3],
  [2, 1],
  [1, 3],
];
```
